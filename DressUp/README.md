# 👕 Dress Up Web App

A small JavaScript project where you can "dress up" a character by selecting different clothing items.

## 🎯 Purpose

Practice DOM manipulation, state management, and user interaction using buttons.

---

## 🖼️ Preview

> Click the buttons to change the character's outfit:
- Hats
- Shirts
- Pants
- Shoes

Each click updates the corresponding image on the screen.

---

## 🚀 Features

- Clothing selection system by category
- State control using variables (`pos1`, `pos2`, etc.)
- Dynamic image updates (`src`)
- Simple UI with interactive buttons

---

## 🧠 Concepts Used

- Basic JavaScript
- DOM manipulation (`getElementById`)
- Events (`onclick`)
- Conditional logic (`if / else`)
- State management
- CSS positioning (`position: absolute`)
